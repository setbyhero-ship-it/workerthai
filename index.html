FAQ

คำถามที่พบบ่อย

ข้อมูลที่กรอกถูกเก็บไว้ไหม?
ไม่ครับ การคำนวณทั้งหมดเกิดขึ้นในเบราว์เซอร์ของคุณเอง ไม่มีการส่งข้อมูลไปยัง server ใดๆ ปิดหน้าต่างแล้วข้อมูลหายทันที
ผลการคำนวณแม่นยำแค่ไหน?
ถูกต้องตามกฎหมายและอัตราที่ประกาศใช้จริง อัพเดทต่อเนื่อง อย่างไรก็ตามเครื่องมือนี้เป็นการประมาณการ กรณีพิพาทหรือตัดสินใจสำคัญควรปรึกษาผู้เชี่ยวชาญด้วย
ใช้บนมือถือได้ไหม?
ได้ครับ ทุก tool รองรับทั้ง desktop, tablet และ mobile ใช้งานได้สะดวกทุกขนาดหน้าจอ
มีค่าใช้จ่ายในการใช้งานไหม?
ฟรีทั้งหมดครับ ไม่มีค่าสมัคร ไม่มีค่ารายเดือน ไม่ต้องใส่บัตรเครดิต WorkerThai ดำเนินการโดยรายได้จากโฆษณา เพื่อให้บริการฟรีแก่ทุกคน
พบข้อผิดพลาดหรืออยากแนะนำ tool ใหม่?
ส่งอีเมลมาที่ contact@workerthai.com ยินดีรับฟังทุกความคิดเห็น และจะอัพเดทแก้ไขให้เร็วที่สุด
